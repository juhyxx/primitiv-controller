# primitiv-controller
## Midi controller with continuous controller

For use with:
* Line 6 pod app
* vst keyboards

## Description

MIDI channel: 1

4 button controller
 - control 41
 - control 42
 - control 43
 - control 44
 
 continues controller (volume/expression pedal)
 - control 1
 - control 41 - fast switch of expression pedal
